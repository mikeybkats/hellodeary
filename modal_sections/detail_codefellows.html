Codefellows Coding Bootcamp 201

My first foray into any kind of real web development started at Codefellows in Seattle. When I say real, what I mean is: I actually began developing my process in a way where I started to build confidence in the projects I was working on.

If I learned anything at Codefellows, it is to take problems one small bite at a time. And, although I am a very new developer I have tackled a wide range of different projects.

Some of the most important things I learned at code school: The document object model, and how Javascript interacts with it. How to use a version control system like git. How to collaborate with other developers by pair programming. Using the command line. Copious amounts of Javascript. It was a great introduction to code for me. I feel that they did a an excellent job at drilling our whole class on the foundational skills. All in all, we completed four to five projects:
